# How do I change the name of a Shared Drive
## Introduction
This guide is meant to be followed by the Managers of shared drives.
If you are not a manager of the shared drive and you want the name changed, please contact the manager(s) and they will have the permission to change the name.
## Step-by-Step Instructions
Go to the shared drive for which you want to change the name.
If you see it on the left panel, under SHared drives, right-click on it and rename it.
If you have the drive open, you can go to the top left corner, click on the name, and then rename.
