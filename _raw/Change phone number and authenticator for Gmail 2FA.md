# Gmail/Google Account 2FA change
Introduction: This short guide provides information on how to change
your number for 2FA in Gmail

## Preparation Steps
1. Request a Google backup code for accessing the account from support by creating a
ticket on the Jira portal https://jira.visma.com/servicedesk/customer/portal/8 .

9. Delete the old phone number and add the new number.

Microsoft) on your phone, scan the code, and you should see a new code appear.
