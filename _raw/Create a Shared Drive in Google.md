# Create a Shared Drive in Google
Date: 13.03.2024
Tags: Google Drive, Shared Drive, Access Management, Drive Creation, Manager Access
This guide provides a step-by-step process on how to create a Shared Drive in Google and manage its access.
## Steps to Create a Shared Drive
Go to Drive: Navigate to your Google Drive.
Right-click on ‘Shared drives’ and select ‘New Shared Drive’.
Name the Shared Drive: In the pop-up window, enter the desired name and click ‘Create’. At this point, you are the only member of the Shared Drive.
If you don’t have access to create a Shared Drive or if the Create Shared Drive option is grayed out, please create a ticket for Service Desk by accessing **support.visma.com**
