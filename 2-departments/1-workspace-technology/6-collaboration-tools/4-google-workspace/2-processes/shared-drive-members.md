# Shared Drive — Add or Remove Members

**Owner:** Google Workspace Team (Collaboration Tools)
**Audience:** Shared Drive Managers
**Last reviewed:** 2026-04-28

---

## Purpose

Allow Shared Drive Managers to add or remove members and assign the correct access role.

---

## Prerequisites

- You must have the **Manager** role on the Shared Drive to make member changes

---

## Steps

1. Open your Chrome browser and navigate to the Shared Drive (use the 9-dots menu in the top-right corner to access Drive)
2. Click on the **name of the Shared Drive**
3. Click **Manage Members**
4. In the new window, type the name or email address of the person to add
5. On the right side of the input box, select the appropriate role

---

## Role Descriptions

| Role | Permissions |
|------|-------------|
| Viewer | View files only |
| Commenter | View and comment on files |
| Contributor | Add and edit files |
| Content Manager | Add, edit, move, and delete files |
| Manager | Manage content, people, and settings |

---

## Escalation

| Issue | Action |
|-------|--------|
| You are not a Manager and need to add a member | Ask an existing Manager to follow the steps above |
| No Manager available or drive has no active Manager | Create a request at support.visma.com |
