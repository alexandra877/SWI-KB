# 2.2.2. Software License Request Process

How to request, purchase, and renew software licenses at Visma.

---

## General Process

```
User needs a license
       ↓
Search MyTools (mytools.visma.com)
       ↓
Found? → Submit request → Manager approves
       ↓                       ↓
   Not found            Provisioned automatically
       ↓                (or Jira ticket created for Support)
Submit ticket at
support.visma.com
       ↓
Ticket routed to Procurement
```

**Primary channel:** [mytools.visma.com](https://mytools.visma.com) — start here for all license requests.

**Fallback:** [support.visma.com](https://support.visma.com) — use when the tool is not listed in MyTools or when a custom purchase is needed.

---

## License Request FAQ

> If your tool is listed below, follow the tool-specific instructions — they take precedence over the general process above.

### GitHub

Submit a request at [support.visma.com](https://support.visma.com).

### Microsoft (CSP/NCE, Office 365)

Submit a request at [support.visma.com](https://support.visma.com) and include your manager's approval.

### Microsoft Teams

Submit a request at [support.visma.com](https://support.visma.com) and include your manager's approval.

### Adobe

Submit a request at [support.visma.com](https://support.visma.com) and include your manager's approval.

### Visual Studio

Submit a request at [support.visma.com](https://support.visma.com). Include "Visual Studio license" in the request title and provide manager approval. Visual Studio licenses are included by default with GitHub Enterprise.

### Docker

Submit a request at [support.visma.com](https://support.visma.com). The Tools team will assist.

### Browserstack, Figma, JetBrains, Productboard

Submit a ticket at [support.visma.com](https://support.visma.com).

### JetBrains (renewal)

Submit a ticket at [support.visma.com](https://support.visma.com). Include "JetBrains license" in the title. Manager approval required.

### Power BI Pro

Submit a ticket at [support.visma.com](https://support.visma.com). Include "Power BI Pro license" in the title. Manager approval required.

### VismaGPT

1. Go to [MyTools.visma.com](https://mytools.visma.com) and log in.
2. Search for **VismaGPT** (one word).
3. On the row "Need help with this resource?", click the arrow and select **Access**.
4. The request goes to your manager — once approved, the license is automatically assigned.

### Snagit / Camtasia (TechSmith)

**Option 1 — via MyTools:**
1. Go to [MyTools.visma.com](https://mytools.visma.com) and log in.
2. Search for **Camtasia** (one word).
3. Click the arrow on "Need help with this resource?" and select **Access**.
4. Specify in the note field whether you need Camtasia or Snagit.

**Option 2 — via Visma Support:**
1. Go to [support.visma.com](https://support.visma.com).
2. Include the product name (Camtasia or Snagit) in the request title.

Manager approval is required in both cases. Once approved, you will receive a license key and download link.

### Azure / AWS / GCP subscriptions

Follow the Cloud Account Request process: https://confluence.visma.com/display/CTO/Understanding+Cloud+Account+Request+Ticket

### Tools not listed in MyTools (Windsurf, CodeRabbit, Khonraad, Cursor, Lovable, others)

If a tool is not available in [mytools.visma.com](https://mytools.visma.com), it is not part of the group agreement. Submit a ticket at [support.visma.com](https://support.visma.com) for Procurement to discuss your options.
