# Procurement & Vendor Management — Processes

This folder contains standard operating procedures for procurement, vendor management, and contract lifecycle workflows.

## Process template

Copy `process-template.md` to create new process documentation. Each process should define:
- Scope and trigger conditions
- Required approvals and decision makers
- Step-by-step workflow
- Compliance and security checkpoints
- Escalation paths

## For agents

Load processes here to understand HOW Visma evaluates, selects, contracts with, and manages vendors.
