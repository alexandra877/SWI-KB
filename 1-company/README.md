# Company

Company-level knowledge: identity, structure, strategy, policies, processes, brand, and decisions.

**Not an employee directory.** Individual team members belong in their department's `3-team.md`.

| Section | Content |
|---------|---------|
| `1-identity/` | Mission, vision, values, positioning |
| `2-structure/` | Org chart + management layer (see README for org_type variants) |
| `3-strategy/` | Current focus, OKRs, roadmap |
| `4-policies/` | AI policy, security, communication, finance rules |
| `5-processes/` | Cross-company processes: onboarding, hiring, procurement, KB governance |
| `6-brand/` | Voice & tone, visual identity |
| `7-decisions/` | Company-level Architecture Decision Records |
